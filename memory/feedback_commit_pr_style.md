커밋 메시지와 PR 설명에 "Generated with Claude Code" 또는 "Co-Authored-By: Claude" 같은 AI 생성 표시를 절대 추가하지 않는다.

**Why:** 사용자가 명시적으로 금지 요청함.

**How to apply:** commit, PR title/description 작성 시 Co-Authored-By 라인, Generated with 푸터 모두 생략.
